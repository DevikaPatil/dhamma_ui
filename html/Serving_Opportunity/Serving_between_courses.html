Between Courses

May you all gain strength in Dhamma for your own benefit, and may you continue to serve others for the good and benefit of so many. -S.N. Goenka

General Guidelines

  • Please adhere to the Code of Conduct while serving at the center between courses. The Dhamma Service Source Book, available online and also located in the server libraries (in the server dorms) is a wonderful resource for growing in Dhamma Service.
  • If you are approved to reside in a student room between courses, please make sure you clean up promptly before the course begins.
  • Please attend the three daily group sittings in the main Dhamma Hall.
  • As noted on the schedule, please meet in the kitchen following the 8 – 9am and 2:30 – 3:30pm group sittings for Dhamma Service.

In the Kitchen

  • If you have any questions about using any equipment, please ask a staff member.
  • Please make sure the ovens and all burners and turned OFF when no one is in the kitchen.
  • All in between servers are responsible for cleaning up after meals are served.

Day -1

  • On the day prior to Day 0, following the 2:30 – 3:30pm group sitting, all available servers can help set up the Dhamma Hall.
